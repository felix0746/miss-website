🔧 登入問題故障排除指南

如果您在登入後台時遇到問題，請按照以下步驟進行故障排除。

常見問題症狀

  • 登入按鈕點擊後無反應
  • 顯示「系統初始化失敗」錯誤
  • 顯示「網路連線失敗」錯誤
  • 登入後無法進入後台
  • 瀏覽器顯示空白頁面

解決方案

方案 1: 清除瀏覽器快取

這是最常見的解決方法，適用於大部分問題。

步驟 1:

在登入頁面點擊「清除快取」按鈕，或手動清除瀏覽器快取。

步驟 2:

重新整理頁面 (Ctrl+F5 或 Cmd+Shift+R)

步驟 3:

重新嘗試登入

方案 2: 檢查網路連線

確保您的網路連線穩定，且可以訪問 Google 服務。

測試 1:

嘗試訪問 Google 確認網路連線

測試 2:

嘗試訪問 Firebase 確認服務可用

測試 3:

如果使用公司網路，確認防火牆沒有阻擋相關服務

方案 3: 使用診斷工具

使用我們提供的診斷工具來檢查系統狀態。

步驟 1:

點擊登入頁面的「🔍 執行診斷工具」連結

步驟 2:

按照診斷工具的指示進行各項測試

步驟 3:

根據診斷結果採取相應的解決措施

方案 4: 瀏覽器特定解決方案

Chrome / Edge
清除快取:

按 F12 開啟開發者工具 → Application → Storage → Clear storage

停用擴充功能:

暫時停用所有擴充功能，特別是廣告阻擋器

Firefox
清除快取:

按 Ctrl+Shift+Delete → 選擇「所有時間」→ 勾選所有項目 → 清除

停用附加元件:

暫時停用所有附加元件，特別是 NoScript 或 uBlock Origin

Safari
清除快取:

Safari → 偏好設定 → 進階 → 顯示開發選單 → 開發 → 清空快取

停用內容阻擋器:

暫時停用內容阻擋器功能

方案 5: 檢查瀏覽器設定

JavaScript:

確保瀏覽器已啟用 JavaScript

Cookie:

確保瀏覽器允許 Cookie

彈出視窗:

允許網站顯示彈出視窗

混合內容:

如果網站使用 HTTPS，確保允許載入混合內容

⚠️ 重要提醒

  • 請確保您使用的是正確的登入憑證
  • 如果問題持續存在，請聯繫系統管理員
  • 不要在多個瀏覽器標籤中同時登入
  • 登入後請勿長時間閒置，系統會自動登出

方案 6: 終極解決方案

如果以上方法都無效，請嘗試以下步驟：

步驟 1:

完全關閉瀏覽器

步驟 2:

清除瀏覽器所有資料（包括密碼、書籤等）

步驟 3:

重新啟動電腦

步驟 4:

使用無痕/隱私模式開啟網站

步驟 5:

重新嘗試登入

如果問題仍然存在

請提供以下資訊給技術支援：

  • 瀏覽器類型和版本
  • 作業系統版本
  • 錯誤訊息的完整內容
  • 診斷工具的測試結果
  • 問題發生的具體時間
